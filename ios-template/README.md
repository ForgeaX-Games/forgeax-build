# ForgeaX iOS Xcode Project (exported)

This directory is a standard Xcode project generated by ForgeaX Studio
("Package Game -> iOS"). The game runs inside a full-screen `WKWebView`, served
from the app bundle's `public/` folder over a custom `forgeax-app://` scheme
(via `ForgeaxSchemeHandler`) with correct MIME types (`application/wasm`, etc.)
so `WebAssembly.instantiateStreaming` and WebGPU/WebGL module loading work
without any network permission.

## Open & build

1. Open `ForgeaxPlayer.xcodeproj` (or the renamed `<project>.xcodeproj`) in
   **Xcode 15 or newer** on **macOS**.
2. Select the `ForgeaxPlayer` scheme and a device / simulator.
3. In **Signing & Capabilities**, pick your Team (Apple ID). A free Apple ID
   works for on-device debugging; a paid Apple Developer account ($99/yr) is
   required for TestFlight / App Store distribution.
4. Run on a device/simulator, or `Product -> Archive` to export an IPA.

> Building an IPA requires macOS + Xcode + a signing identity. ForgeaX Studio
> only *exports the project*; it does not (and cannot off-mac) compile the IPA.

## Integrate third-party SDKs

Add SDKs (payment / ads / login) via Swift Package Manager or CocoaPods and
wire them in `ForgeaxPlayer/`. To call native code from the game JS, add a
`WKScriptMessageHandler` to the `WKWebViewConfiguration` in
`ViewController.swift`.

## Requirements / caveats

- **WebGPU vs WebGL**: WebGPU in `WKWebView` is recent and gated; the ForgeaX
  engine auto-falls back to its WASM + WebGL2 path when WebGPU is absent. WebGL2
  does not require a secure context, so the custom scheme works. If a future
  build needs WebGPU's secure-context guarantee, switch the host to a local
  `127.0.0.1` server (mirroring the Windows launcher) instead of the custom
  scheme.
- **Icon**: `ForgeaxPlayer/Assets.xcassets/AppIcon.appiconset` holds the app
  icon. The exporter writes a single 1024x1024 PNG when provided; otherwise the
  slot is empty and the app ships with a blank icon — set a full icon set via
  Xcode's asset catalog for release.
- **Signing**: the project ships with automatic signing and no Team; configure
  yours in Xcode before building.

## Layout

```
ForgeaxPlayer.xcodeproj/        # Xcode project (target/scheme: ForgeaxPlayer)
ForgeaxPlayer/
  AppDelegate.swift             # UIKit app entry (scene-based lifecycle)
  SceneDelegate.swift           # creates the window + ViewController
  ViewController.swift          # full-screen WKWebView host + scheme registration
  ForgeaxSchemeHandler.swift    # forgeax-app:// -> bundle public/, fixes MIME
  Info.plist                    # display name + orientation (rewritten on export)
  Assets.xcassets/AppIcon...    # launcher icon
  public/                       # the web game (index.html, *.js, *.wasm, assets)
```
